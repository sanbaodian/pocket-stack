# 开发示例

Pocket Stack 的 98% 的特性是 Vibe Coding 开发的结果。为了测试在该实践的基础上AI辅助开发新功能的效果，我开发了一个SAAS模式的任务管理功能。

该示例完全使用AI开发。经过在多个开发Agent测试，虽然效果不同，但都能完成开发。

## 功能特性

- 仪表盘：个人和管理员有不同的仪表盘。
- 任务管理：支持列表、分页、检索、排序、创建、编辑、删除等。
- 视图切换：用户可以在表格和看板之间切换视图。
- 任务看板：看板实现了任务卡片的拖拽更换泳道和排序。
- 任务日历：可以在日历中查看每日创建的任务。
- 任务归档：任务可以设置为归档，在“归档箱”进行管理、恢复。
- 全屏切换：任务管理页面可以全屏展开。
- 前端状态持久化：利用浏览器持久化用户视图设置等数据。
- 数据隔离：不同用户的任务数据隔离。管理员只能看到统计数据。
